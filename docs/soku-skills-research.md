# Soku Skills 调研记录

> 来源：https://soku.ai/skills  
> 调研用途：只做产品结构、命名、分组和 Pro 化灵感参考，不复制 Soku 的 Full Skill 原文。最终对中文用户露出时，统一转为中文业务动作名称。

## 总体观察

Soku 的 Skills 更像营销/GTM/CRO 的“业务动作卡片”，而不是底层工具清单。每个 skill 的页面结构基本包含：

- 这个 skill 是什么
- 适合谁
- 如何工作
- 关键特性
- 什么时候用
- FAQ

这对本项目的启发是：中文 Skills 不应露出为 `PDF`、`SEO`、`Copywriting` 这类技术或能力名，而应露出为“诊断网站”“分析广告”“生成报价包”“压缩老板结论”这类业务动作。

## 对本项目的直接启发

- Skills 页面要强调“谁用、什么时候用、输出什么”，不是只列能力。
- Pro 版本要有清晰边界：官方做通用能力，我们做外贸业务交付。
- 很多 Soku skills 属于 SaaS 增长，不适合直接搬到外贸第一批，但适合第三批 Pro 化时借鉴结构。
- `Search Content Brief`、`AI SEO`、`GEO Visibility Audit` 的组合非常适合转成 `B2B SEO / GEO Pro`。
- `Competitive Battlecard`、`Competitor Alternatives` 的组合适合转成 `外贸竞品研究 Pro`。
- `Paid Ads`、`Paid Keyword Spy`、`Analytics Tracking` 的组合适合转成 `外贸广告分析 Pro`。
