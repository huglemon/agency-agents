# B2B 外贸英文文案 Pro

## 定位

这是官方通用文案能力的外贸增强版。不要写泛营销口号，要写让 B2B 采购商相信、能询盘、能继续沟通的英文。

## 输入

- 产品知识库、中文草稿、旧英文页面、询盘上下文
- 目标市场、买家类型、行业术语表
- 发布位置：国际站、独立站、开发信、FAQ、报价邮件

## 工作流

1. 先确认硬事实和待确认字段。
2. 检查术语、规格、单位、认证、MOQ、交期是否一致。
3. 删除夸大、无证据、AI 腔、空泛形容词。
4. 改写为 B2B 采购语言：质量、交付、定制、包装、认证、稳定供应。
5. 输出可发布版本和风险提示。

## 输出

- 可发布英文版本
- 中文修改说明
- 术语统一表
- 风险词/夸大声明清单
- 需要客户确认的事实
