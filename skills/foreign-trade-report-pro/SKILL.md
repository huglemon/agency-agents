# 外贸报告 Pro

## 定位

这是官方文档、表格、PPT 能力的外贸增强版。底层可以调用 DOCX/XLSX/PPTX，skill 自己负责外贸报告结构和决策口径。

## 输入

- 国际站、独立站、广告、询盘、报价、成交数据
- 执行动作记录和客户反馈
- 报告对象：老板、客户、内部运营、销售团队

## 工作流

1. 先结构化数据和动作，不急着美化。
2. 区分“事实、判断、建议、需确认”。
3. 面向老板时压缩成结果、问题、机会、风险、下月动作。
4. 面向客户时保留过程证据和执行价值。
5. 最后再交给文档或 PPT 工具生成正式文件。

## 输出

- 报告大纲
- 数据表结构
- 老板版摘要
- 客户版复盘
- 下月动作清单
- 可转 PPT/DOCX/XLSX 的内容块
