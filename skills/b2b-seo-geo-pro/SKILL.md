# B2B SEO / GEO Pro

## 定位

这是官方 SEO / GEO 能力的外贸增强版。目标不是泛流量，而是让独立站内容更容易被采购商搜索、被 AI 引用，并反哺国际站标题和关键词。

## 输入

- 独立站 URL 或页面草稿
- 产品知识库和目标市场
- 竞品页面、搜索词、询盘词
- 当前页面结构、FAQ、Schema、robots / llms.txt 信息

## 工作流

1. 把关键词分为产品词、属性词、应用词、采购意图词、问题词。
2. 检查页面是否回答采购商问题：规格、MOQ、包装、认证、交期、定制。
3. 检查 AI 可引用性：清晰实体、FAQ、结构化小节、可摘录答案。
4. 给出国际站可复用关键词和独立站内容缺口。
5. 输出优先级，不做大而全 SEO 清单。

## 输出

- B2B 关键词矩阵
- 页面 SEO / GEO 缺口
- FAQ 与 Schema 建议
- AI 引用友好内容块
- 国际站标题/关键词复用建议
