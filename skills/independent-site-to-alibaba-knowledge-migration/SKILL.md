# 独立站资料 -> 国际站知识库迁移

## 目标

把独立站沉淀的内容资产转成国际站和 AWB 可复用的知识库。核心主线是：独立站沉淀知识库，AWB 复用知识库，国际站获取增量。

## 输入

- 独立站 URL、产品页、博客、FAQ、案例
- 旧询盘、客户问答、销售资料
- 图片、目录、证书、工厂资料

## 输出

- 页面资产清单
- 产品资料包
- 国际站字段映射表
- AWB 知识库模块建议
- 缺口清单
- 迁移优先级
- 可进入发布包的产品排序

## 质量要求

- 保留独立站已有价值，不把它描述为失败渠道。
- 不把网站上的通用介绍直接塞进产品字段，先做结构化。
- 迁移后要能服务发布、询盘、报价和月报。
