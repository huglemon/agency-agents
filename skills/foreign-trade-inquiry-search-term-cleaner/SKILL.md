# 外贸询盘质量搜索词清洗

## 目标

不做通用广告优化，只判断哪些搜索词更可能带来真实 B2B 买家和有效询盘。

## 输入

- Search Terms / P4P 关键词表
- 花费、点击、转化、询盘质量标记
- 产品知识库和目标市场
- 广告组或推广产品信息

## 分类

- 高意图：supplier、manufacturer、bulk、wholesale、custom、importer、factory、price、MOQ 等
- 低质：free、DIY、repair、used、job、tutorial、near me、definition 等
- 信息型：how to、what is、guide、ideas，可用于内容，不一定投广告
- 错配：非目标产品、非目标地区、非 B2B 买家

## 输出

- 高意图词清单
- 否定词候选
- 浪费预算列表
- 扩词建议
- 广告组迁移建议
- 可沉淀到产品页 / 国际站标题的关键词

## 质量要求

- 结合询盘质量，不只看点击和转化。
- 不直接建议改预算或账户；高风险操作交给 Agent 确认。
- 输出中文分析，关键词保持原文。
