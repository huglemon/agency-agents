# 外贸广告分析 Pro

## 定位

这是官方广告分析能力的外贸增强版。重点不是 CTR、CPC 本身，而是广告是否带来真实 B2B 买家和可推进询盘。

## 输入

- Google Ads / P4P 数据
- Search Terms、关键词、广告组、花费、点击、转化
- 询盘质量标记、报价结果、成交反馈
- 产品知识库和目标市场

## 工作流

1. 先按 B2B 采购意图清洗搜索词。
2. 把花费和询盘质量放在一起看。
3. 区分高意图词、低质词、信息词、错配词。
4. 找出可沉淀到独立站和国际站 Listing 的关键词。
5. 输出老板能理解的预算动作建议。

## 输出

- 搜索词质量分类
- 否定词候选
- 浪费预算清单
- 高意图词和扩词建议
- 产品页/国际站标题复用建议
- 下月广告动作建议

## 安全边界

不要直接执行预算、出价、关键词修改。高风险变更必须交给 Agent 做确认。
