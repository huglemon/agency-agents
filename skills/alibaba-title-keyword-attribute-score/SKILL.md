# 阿里产品标题 / 关键词 / 属性评分

## 目标

专门评估国际站产品字段质量，不做泛 SEO。输出评分、风险和可改版本。

## 评分维度

- 标题清晰度：是否一眼知道产品是什么
- 搜索匹配度：是否覆盖核心采购词和属性词
- 买家意图：是否体现 wholesale、supplier、custom、bulk、manufacturer 等 B2B 语义
- 属性完整度：规格、材质、尺寸、包装、用途、认证是否齐
- 堆词风险：是否重复堆砌无关词
- 转化支持：是否能引导买家继续看详情或询盘

## 输出格式

- 总分：0-100
- 分项评分
- 主要问题
- 修改建议
- 建议标题 3 个版本
- 关键词补充表
- 属性缺口表

## 质量要求

- 不用“爆款”“最优”等无依据词。
- 只基于资料和合理行业词，不把竞品词当事实。
- 中文解释，字段建议可给英文。
