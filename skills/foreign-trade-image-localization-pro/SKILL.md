# 外贸图片翻译 / 本地化 Pro

## 定位

这是图片翻译能力的外贸增强版。重点不是把字换成另一种语言，而是让图片仍然适合 B2B 采购场景和目标市场。

## 输入

- 产品图、详情页卖点图、包装图、Banner、海报
- 源语言和目标语言
- 目标市场、买家类型、平台位置
- 必须保留的品牌、型号、数字、单位、认证

## 工作流

1. 识别所有可见文字和必须保留元素。
2. 翻译为采购商自然表达，避免夸大承诺。
3. 检查数字、单位、型号和认证是否被误改。
4. 根据文字长度调整字号、换行、对齐和文本框。
5. 输出编辑 prompt 或交给图片编辑 skill 执行。

## 输出

- 翻译文案表
- 保留元素清单
- 风险词和夸大声明
- 图片编辑 prompt
- 版式适配建议

## 来源边界

可参考 Humanizer-zh 的自然化原则和 guizang-social-card-skill 的版式 QA 思路；不复制第三方模板或全文。
