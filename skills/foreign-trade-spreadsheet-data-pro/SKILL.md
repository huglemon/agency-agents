# 外贸表格数据 Pro

## 定位

这是官方表格能力的外贸增强版。它不替代 XLSX 工具，而是定义外贸运营表应该怎么清洗、归因和复盘。

## 输入

- 产品发布表
- 询盘表
- 报价表
- 样品表
- 成交表
- 搜索词表
- 月度复盘表

## 工作流

1. 统一字段名称和日期、产品、客户、国家、来源。
2. 建立漏斗：曝光/点击 -> 询盘 -> 报价 -> 样品 -> 成交。
3. 标记产品机会、低质询盘、预算浪费和资料缺口。
4. 输出可给老板看和可给运营执行的两层结论。

## 输出

- 字段清洗建议
- 漏斗指标
- 产品机会表
- 询盘质量分层
- 下月动作表
- 缺失数据清单
