# 国际站经营月报结构化

## 目标

把国际站运营数据整理成月报骨架，再交给文档、表格或 PPT 工具生成正式交付件。

## 输入

- 曝光、点击、访客、询盘、RFQ、P4P 数据
- 产品发布和优化动作
- 主推品、询盘质量、报价进度
- 本月问题和下月目标

## 输出

- 本月概览
- 产品表现
- 流量和曝光
- 询盘和 RFQ
- P4P 花费与问题
- 发布/优化动作
- 机会产品
- 风险和缺口
- 下月动作清单

## 质量要求

- 先结构化，不急着写漂亮话。
- 数字缺失时标注缺口，不编环比。
- 可接 `boss-facing-business-summary-compressor` 生成老板版摘要。
