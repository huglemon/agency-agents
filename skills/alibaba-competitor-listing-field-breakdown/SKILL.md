# 阿里竞品产品页字段拆解

## 目标

只拆国际站产品页字段和成交资产，不做泛竞品战略报告。

## 输入

- 竞品国际站产品链接
- 竞品标题或截图
- 当前产品知识库
- 目标市场和类目

## 输出

- 标题结构
- 关键词与属性词
- MOQ / 包装 / 认证 / 价格线索
- 主图和详情页模块
- 信任背书
- 内容缺口
- 可借鉴点
- 不应复制或风险点

## 质量要求

- 不复制竞品文案。
- 只把竞品信息作为参考，不写成当前产品事实。
- 输出必须能反哺标题、属性、详情页和素材需求。
